# ⚡ 柱塞泵节能测算 & 投标文件生成

专业级柱塞泵节能测算工具，支持输入参数自动计算节电率、年节电量、投资回收期，并可一键生成投标文件（Word格式）。

## 功能特点

- 📊 **智能节能测算** — 输入原设备运行参数，自动计算综合节电率
- 💰 **投资回收分析** — 自动分析投资回收期和5年净收益
- 📝 **一键投标文件** — 自动生成完整投标文件（含技术方案、报价表、资质证明等）
- 📱 **测算历史** — 支持登录后保存和查看历史测算记录
- 🖨️ **打印导出** — 支持打印/导出测算报告

## 使用方式

直接在浏览器中打开 `index.html` 即可使用，无需安装任何依赖。

## 技术栈

- HTML5 + CSS3 + JavaScript（纯前端单页应用）
